# 安全说明 / Security

Imago 在设计上避免了此前 PHP 版本的所有已知漏洞。

## 防护措施

| 风险 | 防护 |
|------|------|
| SQL 注入 | GORM 参数化查询，无原生 SQL 拼接 |
| 会话劫持 | JWT 替代会话 Cookie，支持过期和黑名单 |
| 路径遍历 | `realpath()` 前缀边界校验 |
| Passkey 伪造 | go-webauthn 完整签名验证 + 服务器挑战码 + Origin 校验 |
| CSRF | Token 机制覆盖管理端状态变更操作 |
| 暴力破解 | 按 IP 速率限制 |
| XSS 上传 | SVG 文件完全拒绝上传 |
| 默认密钥 | 安装后配置为空，需手动设置 |

## 报告漏洞

如发现安全漏洞，请通过 GitHub Issues 报告，不要在公开评论中透露漏洞细节。
